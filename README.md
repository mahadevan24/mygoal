# 🎯 MyGoal

A dedicated, highly visual 1-year preparation tracker and motivational dashboard designed to help you land a software engineering role at top-tier product companies (like Google, Amazon, and Microsoft) by July 2027.

---

## 📌 Vision & Objective

To spend **3 hours daily**—consistently, with discipline, and without fail—for one full year, divided equally among the core pillars of software engineering interviews. 

*   **Timeline:** July 1, 2026 ➔ July 2027 (1 Year)
*   **Daily Commitment:** 3 Hours (1 Hour DSA + 1 Hour LLD + 1 Hour System Design)
*   **Ultimate Goal:** Get offers from Tier-1 Product Companies (Google, Amazon, Microsoft, etc.)

---

## 🚀 Core Features

### 📅 LeetCode-Style Calendar View
*   **Visual Contribution Grid:** A color-coded calendar grid (similar to GitHub or LeetCode contribution graphs) representing the full 365-day journey.
*   **Goal Status Coloring:**
    *   🟩 **Green:** 3 hours completed (1 hr DSA + 1 hr LLD + 1 hr System Design).
    *   🟥 **Red/Orange:** Target missed (less than 3 hours total or any topic skipped).
    *   ⬜ **Gray:** No progress logged yet.
*   **Hover Details:** Displays specific hours logged per topic on any selected date.

### ⚡ Streak & Progress Tracker
*   **Current Streak:** Track consecutive days of meeting the 3-hour goal.
*   **Longest Streak:** Visual record of your personal best to keep you motivated.
*   **Total Hours Logged:** Accumulative statistics for DSA, LLD, and System Design to see your long-term growth.

### 🖼️ Interactive Vision Board
*   **Motivation Canvas:** A customizable visual space to upload and pin inspiring images (e.g., target company logos, dream workspaces, tech stacks).
*   **Daily Quote / Reminder:** Display customizable daily affirmations and countdowns to the July 2027 target date.

### ⏱️ Topic Timer & Logger
*   **Focused Timer:** Built-in stopwatch/pomodoro timer for each individual session (DSA, LLD, System Design).
*   **Daily Checklist:** Mark off each study category as you complete your daily 1-hour sessions.

---

## 🛠️ Tech Stack

*   **Framework:** Next.js (App Router, React)
*   **Styling & UI:** Tailwind CSS & Shadcn/ui Components
*   **Database & Storage:** Supabase (PostgreSQL for user metrics/logs, Supabase Bucket Storage for vision board images)
*   **Authentication:** Supabase Auth (Secure user sign-up/login, session persistence)
*   **Charts & Visuals:** Tailwind-styled custom SVG contribution grid (LeetCode-style)

| Topic | Focus Area | Daily Target |
| :--- | :--- | :--- |
| **DSA (Data Structures & Algorithms)** | Problem-solving patterns, LeetCode style exercises, complexity analysis. | 1 Hour |
| **LLD (Low-Level Design)** | Object-Oriented Design patterns, SOLID principles, schema design, and clean code. | 1 Hour |
