# Antigravity Agent Guidelines

These guidelines are customized for this project to ensure seamless collaboration between the Antigravity agent and the user.

## Browser Verification
- **Do NOT open the browser** (e.g., via the browser subagent or any automated browser tools) for verification or interaction.
- Instead, **instruct the user to manually verify** the web page or changes in their own browser.

## Database & Supabase Interaction
- **Do NOT attempt to connect directly** to the Supabase database.
- Do NOT run database migrations or execute queries automatically on the database.
- Instead, when database changes or migrations are needed:
  1. Generate the exact SQL queries or migrations required.
  2. Present these SQL queries to the user.
  3. **Instruct the user to run the queries manually** in their Supabase dashboard or SQL editor.
